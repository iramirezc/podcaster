## Installation

In the project directory, run:

```sh
npm install
```

## Development

To run the app in development mode:

```sh
npm run start
```

## Production

To build the app for production:

```sh
npm run build
```

## Storybook

To see the components catalog:

```sh
npm run storybook
```

## Tests

To make sure everything still works:

```sh
npm run test
```
